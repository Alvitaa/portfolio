# Juan Alva's Portfolio

This is the GitHub repository for the React + Vite and Tailwind CSS app of my Portfolio. This portfolio is designed in such a way that evokes the immersion of being in a subway train station, from entering the station up to being in the train platform.

Here you can know more about me, catch some random facts and have my socials in which you can contact me! Also, the website showcases some of my personal and academic projects I’ve made ever since I started developing with React JS.

## The Thought Process

As I stated in previous projects, I’ve always had such curiosity about trains and urban transport developments so as one of my few interests I decided to design it with that theme in mind.

I had to go outside my comfort zone as I needed to model and render 3D stairs of a station to fit the design of a subway station. In addition, I made the information show inside posters mimicking public notices seen in real life stations. Finally, I made the projects section as if it were a train platform, making the projects be the advertisements and have the user cycle through them as if they were walking by them.

This project has been one of my favorites as I had so much fun making and implementing the design.

## How to run it yourself

1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Run the development server with `npm run dev`
4. Open your browser and navigate to `http://localhost:5173`

Be sure to have installed `node`
